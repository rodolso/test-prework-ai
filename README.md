# 🚀 ¡Bienvenid@s al Prework de Python del Bootcamp AI Engineering!
### The Bridge

¡Estás a punto de dar tus primeros pasos en el mundo de la IA! Todo el material del prework está listo para usarse directamente en el navegador, sin instalar nada. Solo necesitas un par de clics para tener tu propio entorno de programación funcionando. 🎉

---

## ¿Cómo empiezo?

**1. Pulsa el botón de abajo**
El botón *Open in GitHub Codespaces* te llevará a la pantalla de configuración.

**2. Haz clic en el botón verde "Create codespace"**
No cambies ninguna opción, simplemente confirma con el botón verde. ¡Fácil!

**3. Espera unos minutos ☕**
El entorno se configura solo la primera vez. Cuando esté listo, abre la carpeta `notebooks/` y ¡a programar!
